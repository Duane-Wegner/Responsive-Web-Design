Håkon Wium Lie - The Father of CSS

Håkon Wium Lie is a Norwegian web pioneer who created Cascading Style Sheets (CSS) in 1994 while working at CERN. CSS transformed the web by enabling the separation of content from presentation, giving designers and developers the tools to create visually rich and consistent user experiences.

He served as CTO of Opera Software and continues to advocate for web freedom and open standards. His contributions have shaped the modern internet and empowered generations of web developers.
